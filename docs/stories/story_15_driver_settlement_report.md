# User Story: Driver Settlement Report

**As a** Fleet Manager or Accountant,
**I want** to run a driver settlement report for a specific pay period,
**so that** I can accurately calculate driver pay and ensure timely payments.

---

### Acceptance Criteria:

- A "Settlements" or "Payroll" section is available in the web application.
- The user can select one or more drivers and a specific date range for the report.
- The generated report details all completed orders for the selected period.
- The report includes itemized pay calculations (e.g., per mile, percentage of load), deductions (e.g., fuel), and reimbursements (e.g., tolls).
- The report shows a final net pay amount for each driver.
- The user can export the settlement report as a PDF or CSV file.
- The user can mark a settlement as "Paid" to finalize the record.
